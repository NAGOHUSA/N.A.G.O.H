Privacy Architecture

Your data never
leaves your phone.

We built NAGOH with a simple principle: the developer using our app shouldn't have to trust us with their data. So we engineered it so you never have to.

🔑

Keychain-Protected Credentials

Your App Store Connect API keys are stored in the iOS Keychain — the same hardware-backed security that protects your passwords and payment info.

📱

On-Device Storage Only

Sales data, app names, revenue figures — everything is stored locally using SwiftData. Nothing is transmitted to or stored on any server we operate.

🍎

Secure Setup

Add your App Bundle ID to NAGOH. NAGOH creates a unique device-specific push notification URL that you add to your App Information on Apple Connect. That's it!

🚫

No Analytics. No Tracking.

We collect zero analytics about how you use the app. No crash reporters that phone home. No behavioral tracking. Nothing. Your usage is yours alone.
